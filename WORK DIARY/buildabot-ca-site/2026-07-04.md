## 🎯 Decisions Made

- **Architecture:** Single Vite app with public landing + authenticated dashboard — one codebase, one domain, one auth system
- **Color palette:** Violet #7c3aed primary, Yellow-300 #fde047 highlight, Navy/slate text, Violet-tinted bg
- **No teal** — user explicitly rejected it
- **No heavy animations** — Motion for micro-interactions only
- **Mobile-first** sidebar collapses on small screens
- **Telegram-first CTA** — primary action is "message RE-Ai on Telegram", not "sign up"
- **Combined project** — landing page and control center = one website
- **Vite v8** needed esbuild installed separately

## 🔄 Things That Changed Direction

- **Teal → Violet:** User wasn't sold on teal, went through 4 alternatives
- **Gold → Yellow-300:** User said the yellow was "too gold," wanted a more natural yellow
- **shadcn init failed:** Had to write UI components manually (button, card, badge, etc.)
- **Vite v8 build issue:** Required explicit esbuild dependency

## 🧱 What's Blocked

- Better Auth backend not wired yet (needs Hono API server + DB schema)
- No user authentication — dashboard, login, and register pages are static
- No real Postgres data — leads table uses hardcoded sample data
- buildabot.ca domain not purchased yet
- No deployment target configured
